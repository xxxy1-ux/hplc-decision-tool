# HPLC Streamlit Decision Tool

Upload these files to the ROOT of your GitHub repository:

```text
streamlit_app.py
requirements.txt
runtime.txt
README.md
```

Streamlit deploy settings:

```text
Repository: xxxy1-ux/hplc-decision-tool
Branch: main
Main file path: streamlit_app.py
```

The `runtime.txt` file forces Streamlit Cloud to use Python 3.11.9.
