# HPLC Streamlit Decision Tool

This is a GitHub-ready Streamlit web app for an HPLC troubleshooting decision tool.

## Files

Your repository root should contain:

```text
streamlit_app.py
requirements.txt
README.md
```

## Run locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Deploy on Streamlit Cloud

Fill the deployment form like this:

```text
Repository: xxxy1-ux/hplc-decision-tool
Branch: main
Main file path: streamlit_app.py
```

## Important

Do not use `app.py` for Streamlit Cloud.

Streamlit Cloud needs:

```text
streamlit_app.py
```
